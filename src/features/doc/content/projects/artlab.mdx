## Features

- **Text-to-Image Generation** — Generate images from any text prompt
- **Fast & Responsive UI** — Built for speed and usability
- **Dark Mode Support** — Full theme support
- **Clean Design** — Tailwind CSS and shadcn/ui components
- **High-Resolution Output** — AI-generated images in high quality

---

## Tech Stack

- **Framework**: Next.js — React framework for production
- **Styling**: Tailwind CSS — Utility-first CSS framework
- **UI Components**: shadcn/ui — Beautifully crafted components
- **AI Model**: Gemini API — Google's text-to-image model
